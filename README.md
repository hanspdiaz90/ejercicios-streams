# Ejercicios Java - Streams y Lambdas

Este repositorio contiene una colección de ejercicios prácticos desarrollados en Java 11, enfocados en el uso de Programación Funcional, Streams API, Expresiones Lambda y Collectors.

Los ejercicios están organizados en nivel básico e intermedio, utilizando dos clases principales:

- Empleado
- Pedido

Nivel Básico

Operaciones fundamentales con Streams:
- Filtrado con filter() y Predicate.
- Transformaciones con map() y Function.
- Ordenamientos con sorted() y Comparator.
- Conteo y validaciones con count(), anyMatch().
- Extracción de propiedades.
- Uso de referencias a métodos.
- Uso de limit() para paginación simple.

Nivel Intermedio

Operaciones avanzadas con Collectors y composición funcional:
- Agrupación de datos con groupingBy()
- Particionado con partitioningBy()
- Transformaciones con mapping()
- Estadísticas con summarizingDouble()
- Promedios con averagingDouble()
- Sumas con summingDouble()
- Uso de Optional
- Composición de collectors (collectingAndThen)
- Manejo de valores nulos con Optional.
